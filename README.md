# i-am-learning-coding

### Memo
- Including all.
  ```
  #include <bits/stdc++.h>
  ```
- Using directive．
  ```
  using namespace std;
  ```
- Making file.
  ```
  $ echo -e '#include <bits/stdc++.h>\nusing namespace std;\n\nint main(){\n    \n    \n    \n    return 0;\n}' > main.cpp
  ```

- Compiling.
  ```
  g++ main.cpp -std=c++14
  ```
